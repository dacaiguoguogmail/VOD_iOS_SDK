  finishBlock

当访问locatime成功，解析xml成功并localtime值有效时，回调该block。 当访问失败，解析失败或loclatime无效等错误逻辑时，调用 [[NSDate date] timeIntervalSince1970] 获取loclatim，然后回调该block。 总之，如果该block被回调，那么传递的localtime值一定是可用的。
