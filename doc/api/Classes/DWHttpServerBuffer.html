DWHttpServerBuffer.m

Overview

buffer 管理器，用来管理 client <-> DWHttpServerConnection <-> DWServerProxy 之间的交互数据。

DWHttpServerConnection 接收 DWServerProxy 的数据存入 buffer中，形成一个缓存数据链。 DWHttpServerConnection 将数据链中的数据发送给 client。
