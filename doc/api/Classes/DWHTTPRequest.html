  unCopyResponseBody

若设置为YES，则在 -connection:didReceiveData: 中不会copy接受的数据。 默认为NO，若使用 -startAsynchronousWithDestinationFilePath:，则接收到的数据会被写入文件，否则追加到buffer。

@property (assign, nonatomic) BOOL unCopyResponseBody

Discussion

若设置为YES，则在 -connection:didReceiveData: 中不会copy接受的数据。 默认为NO，若使用 -startAsynchronousWithDestinationFilePath:，则接收到的数据会被写入文件，否则追加到buffer。
